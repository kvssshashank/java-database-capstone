# User Stories

## Admin
* As an Admin, I want to add new doctors to the system so that patients can book them.
* As an Admin, I want to view all user accounts to manage system access.

## Doctor
* As a Doctor, I want to view my daily appointments to organize my schedule.
* As a Doctor, I want to write prescriptions for my patients.

## Patient
* As a Patient, I want to search for doctors by specialty to find the right care.
* As a Patient, I want to book an appointment with a doctor for a specific date.
